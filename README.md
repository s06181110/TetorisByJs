# TetorisByJs

## Requirements
- Node & npm

## Installation
```
$ git clone https://github.com/s06181110/TetorisByJs.git
```

## Features
- Implement pvp with socket.io

## Quick Start
Move to installed directory:
```
$ cd TetorisByJs
```
Install packages:
```
$npm install
```
Start the server:
```
$ npm start
```

## Develop
Start by Nodemon:
```
$ npm run dev
```
